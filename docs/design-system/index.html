Design System Draft

CGintegra com visual institucional, claro e rastreavel.

Esta vitrine traduz o momento atual do produto em uma linguagem de interface pronta para dashboard, editor documental, revisao, aprovacao e auditoria. A base cromatica segue a referencia de cgguarulhos.com.br, com vermelho forte, neutros quentes e azul profundo para estrutura.

Direcao

Produto documental com cara de governanca moderna: confiavel, legivel e objetivo, sem perder calor visual nem hierarquia de estados.
